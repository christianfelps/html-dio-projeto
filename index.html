Introducao do html na Pratica

Voltar

Html tem 31 tags basicas para começar a propria a html, head e body estão envolvidas dentro da tag html. Para fazer um titulo so usar a tag h1 e essa tag vai ate o h6 pois vai de titulo subtitulo e assim por diante. Html > a tag que da inicio e não se deve esquecer dela.
head > o que a pagina ira carregar antes de exibir para o usuario.
body > Tudo que esta envolvido nele ira mostrar ao usuario.

Algumas tags são utilizadas para formatar o texto o design de algumas palavras sem o css, tem tags simples "i" para deixar seu texto em italico
